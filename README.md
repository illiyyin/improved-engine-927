
# Project Wellness

Project untuk latihan teman-teman bootcamp kelas react-native.



## Run Locally

Clone the project

```bash
  git clone git@github.com:illiyyin/improved-engine-927.git
```

Go to the project directory

```bash
  cd improved-engine-927
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
